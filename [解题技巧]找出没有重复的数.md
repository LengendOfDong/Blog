# [解题技巧]找出没有重复的数

题目：<br/>
给你一组整型数据，这些数据中，其中有一个数只出现了一次，其他的数都出现了两次，让你来找出一个数 。<br/>
数据如下：

```
1,2,3,4,5,4,3,2,1

```

使用异或方式解题：

解题思路：

```
1^2^3^4^5^4^3^2^1  = (1^1)^(2^2)^(3^3)^(4^4)^5 = 0^0^0^0^5 = 5

```

代码：

```
 public int findNum(int[] array){
        int tmp = 0;
        for(int num : array){
            tmp = tmp^num;
        }
        return tmp;
    }

```

时间复杂度为O(n)，空间复杂度为O(1)
